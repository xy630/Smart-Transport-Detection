# Smart-Transport-Detection
## 项目概述
本项目是一套基于深度学习的智慧交通监测系统，采用 YOLOv8 目标检测与实例分割技术，实现对交通场景的实时感知与分析。系统具备目标检测、车道线识别、交通流量统计、危险行为预警等核心功能，提供友好的Web交互界面

## 技术栈
- 深度学习 ：Ultralytics YOLOv8

- 前端 ：HTML, CSS, JavaScript
- 编程语言 ：Python 3.x
- ##  使用方式
1. 启动后端： python app.py
2. 启动前端服务器： python -m http.server 8000
3. 访问页面： http://localhost:8000/detect_page.html
4. 上传视频，选择检测功能，点击开始检测
